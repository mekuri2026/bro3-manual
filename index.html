ルート捲り & 施設削除
操作マニュアル

マップのマスを順に捲る／鹵獲する「ルート捲り」と、拠点の施設を自動削除する「施設削除」。この2本の使い方をまとめました。パネルの色分けとこのマニュアルの色は対応しています。

はじめに ・ 3ステップで始める

ルート捲りは全体地図で使い、施設削除は都市画面で使います。まずはめくりを動かしてみましょう。

  1. 全体地図を開き、右上の ≡ ルート捲り を押してパネルを開く。
  2. 🔵 めくり ブロックで この拠点の武将を取得 → 使う武将に ☑（自動保存）。
  3. ルート作成: ON にしてマップのマスを順にクリック → 開始。
自己責任自動出兵は規約でBOT行為とみなされる可能性があります。BAN・規約リスクは利用者自身の責任で。長時間の無人放置ほどリスクは上がります。ツールは手動出兵と同じリクエストを送るだけで、検出回避はしません。
最初の一度だけ作り替えた武将選択は出兵に直結します。長く回す前に、少量で「意図した武将・拠点で出るか」を必ず確認してください（武将の排他参照）。

🔵 めくり（ルート捲り）

単騎でマスを順に捲る（＝取得する）機能。武将の選択もこのブロックで行います。

手順

  1. 出兵拠点 を選ぶ（★は現在地）。無ければ ↻ で本拠地含む全拠点を再取得。
  2. この拠点の武将を取得 → 出てきた武将に ☑。複数チェックでローテーション。☑した武将が自動保存され、めくり武将になります（登録ボタンは不要）。各カードにスキル名（例 天帝檄令LV10）が表示され、アイコンをクリックすると全ステータスの詳細が開きます。主力と補助の見分けに使ってください（コストが読み取れるカードは緑のコストバッジも表示）。
  3. ルート作成: ON → マップのマスを順にクリックして経由地を作る。1つ戻す／クリアで修正。破棄チェックをまとめて付けるなら 全経由地の破棄を一括✓。
  4. ルート確認 で赤=取得予定・黄=等級超で止まる可能性・灰=回避、を確認 → 開始。

🛡 番人モード

設定したルート上のマスを見張り、空き地になったら自動で取り直す機能。囲い（リング）を維持したいときに。マスの破棄タイマーを読んで賢く待ちます（完了がまだ遠い間はほとんど確認せず、完了間際だけ細かく確認）。放置OK・リロードしても継続します。

  1. 維持したいマスを経由地クリックで2点以上セット（リング生成で一気に作れます）。
  2. 最短間隔秒（既定10）を決めて 番人モード: ON。これは確認間隔の下限で、普段はもっと疎に待ちます。

めくり・鹵獲と同時に動かせます。内部で出兵を1件ずつ順番に処理するので、出兵元拠点の取り違えは起きません。一時的な読み取り失敗で勝手にOFFになることもありません（OFFにするのは停止ボタンだけ）。

⭕ リング

指定した中心の周りを一周する経由地（リング）を自動生成します。拠点の隣接を囲って守るときに。

  1. 中心を地図で指定 を押し、守りたい拠点のマスをクリック（または 中心 に座標を手入力）。
  2. 距離（中心から何マス外か）を決めて リング生成。
  3. ルート確認 → 開始で捲る。維持したいなら 番人モード をON。

サイクル状況を表示（読むだけ）

サイクル状況を表示 を押すと、サーバー時刻から今のフェーズ（日中／夜間確保／休戦）を判定し、リング各マスの種別（自軍・味方・空き・敵）と保護の残り時間・保護が切れる時刻を一覧表示します。何も実行しない確認専用です。リング上に敵マスがあると警告が出ます（突破口になるため）。

⏱ 逆算スケジュール破棄（v5.57）

「この時刻に破棄が完了してほしい」から逆算して自動で破棄を開始し、破棄が完了して空いた瞬間に同じマスを最速で取り直す機能です。深夜〜早朝の操作をツールが代行します。

  1. リングを生成（または経由地をクリック）。破棄されるのはその中の自軍マスだけです（味方・敵・空きは対象外）。
  2. めくりブロックで再取得に使う武将に ☑（速い武将1体を推奨）。
  3. 目標完了 にサーバー時刻で入力（例 6:00。休戦時間 1:00〜6:10 内を推奨）。領地統制 スキルがあればチェック（所要 3h25m。無ければ 6h）。
  4. 逆算破棄を予約。開始時刻（=完了−所要）になると数秒ずつずらして順に破棄が始まります。
  • 破棄完了は各マスの実際のタイマーを読んで追従します。完了が近づくと細かく監視し、空いた瞬間に同じマスへ出兵→確保を確認してから次のマスへ（空き地は常に最大1マスしか生まれません）。
  • 予約はリロード・画面移動しても保持されます。解除は同じボタン（予約解除）か ■ 全停止。
  • 破棄開始時に自軍でなくなっていたマスは破棄せずスキップします。完了時刻が休戦時間の外だと警告が出ます（空いた瞬間に敵に捲られるリスクがあるため）。

🏴 鹵獲（2チャンネル）

空き地へ武将を出して資源を持ち帰り、帰還したら自動で再出兵する周回機能。A・Bの2系統を別々の拠点・武将で同時に回せます（A=本拠地/塔、B=鹵獲用拠点など）。めくりと並行して動かせます。

A・Bそれぞれで設定

  1. 拠点 を選び 武将取得 → 使う武将に ☑（自動保存。そのチャンネルの鹵獲武将になります）。
  2. 鹵獲先を追加 を押し、マップの空き地をクリック（複数可）。

共通設定と開始

  • 時間(分/最大60)：この時間で自動停止。上限は60分です。
  • 余裕(秒/乱)：再出兵の間隔に足す余裕（乱数）。
  • 周回：ONで帰還ごとに繰り返し。
  • 鹵獲開始(A+B) でA・Bまとめて開始。残り時間と獲得資源（今回／累計）が表示されます。
上限60分長時間の連続出兵はリスクが高いため、1回の鹵獲は最大60分に制限しています。続けたい時は様子を見て再開してください。

武将の排他ルール

重要

同じ武将は同時に2つの用途では使えません（1体は1回しか出兵できないため）。ロックは実行中だけかかります。停止中は☑を自由に付け替えできます（カードの上げ下げはゲーム標準の操作でOK）。

  • 鹵獲を実行中：その武将はめくりのリストで 🔒鹵獲中 となり選べません。
  • めくり／番人を実行中：その☑武将は鹵獲のリストで 🔒他で使用中 となり選べません。

機能の同時利用

めくり・番人・鹵獲A/Bは同時に動かせます。それぞれ別の武将を割り当ててください（同じ武将を使うと帰還待ちで交互に待たされます）。まとめて外すなら、めくりは [全解除]、鹵獲は [武将クリア]。

確認作り替えた武将選択は出兵に直結します。長く回す前に、少量で「意図した武将・拠点で出るか」を確認してください。

🧱 施設削除（別ツール）

拠点の施設を「残す数／壊す数」で指定して削除するツール。都市画面の拠点名ヘッダ横に出る [施設削除] を押すとパネルが開きます（全体地図など都市以外では出ません）。

  1. 都市画面で [施設削除] をクリック。
  2. 施設ごとに 残す数 を入力（チェックを入れると壊す対象が最低1になります）。
  3. 自動巡回を開始。削除はタイマー式で1拠点ずつ順番に進みます（1件あたり数十分〜1時間）。
  4. 対象を変えたい時は 停止 → チェックを付け替え。停止で今の設定はリセットされ、チェックし直した施設が新しい対象になります。
併用注意これはオートビルダーの自動削除を切り出した代替ツールです。オートビルダーの自動削除と同時に動かすと施設を取り合うので、削除を動かすのはどちらか片方だけにしてください。

更新のしかた

ツールはGistで配布され、自動更新に対応しています。手早く最新にするには：

  1. Tampermonkeyのダッシュボードを開く。
  2. 最後の更新をチェック を押す → 新しいバージョンがあれば自動で入れ替わります。ゲーム内に保存した設定は消えません。
URLの見分け末尾が /raw/ファイル名.user.js のURL＝常に最新（自動更新）。/raw/長い英数字/… が挟まるURL＝その時点で凍結。凍結URLを開くと古い版が入り直すので開かないこと。

中身を直したのにバージョン番号が同じだと「最後の更新をチェック」で拾われません。番号が上がっていれば更新されます。

停止・困ったら

  • すべて止めたい：パネル上部の ■ 全停止。めくり・番人・鹵獲をまとめて停止し、取得済みマスは破棄して終了します。
  • ランチャーが消えた：パネル生成でエラーが出た可能性。ページをリロード → それでも出なければ状況を伝えてください。
  • 出兵拠点に本拠地が無い：↻ を押すと本拠地含む全拠点を取り直します（★＝現在地が先頭）。
  • 意図しない武将が出る：武将の排他を確認。めくり／鹵獲の☑が正しいか、長時間動かす前に少量でテスト。
  • ログ：📜ログブロックに動作履歴。ログをコピーで共有できます。
  • 放置するとタイムアウトで止まる：🔧ツールブロックの セッション維持（既定ON）が約3分ごとに軽い通信を送り、放置してもセッションが切れにくくなります。万一切れた時はログに ⚠⚠ セッション切れの可能性 と警告が出るので、ページを開き直してください。
継続の仕様めくりの前進・鹵獲の周回・番人モードは、画面遷移やリロードでも裏で自動再開します。完全に止めたい時は各ブロックの停止か ■ 全停止 を使ってください。
